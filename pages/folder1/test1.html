# 什么是平衡二叉树

平衡二叉树是改进的二叉搜索树,拥有自平衡特性。

# 平衡因子

平衡因子是某节点的左右子树的高度差

# 结论

平衡二叉树中不存在平衡因子大于1的节点

why?

当向平衡二叉树中插入一个新的节点导致树中存在节点不满足二叉平衡性(即某个节点的平衡因子大于1,则以该节点为根的子树，即为最小失衡子树）需要进行平衡二叉树的调整。

# 调整操作

  • LL型

    A的平衡因子>1，并且失衡原因出在A的左孩子的左子树。这个时候我们进行LL调整，B往上抬，替代A的位置，A顺次往下，作为B的右孩子，这个过程中B的原右孩子不得不与B拆散，并且填补到唯一的空缺：A的左子树

    上述操作即为右旋.

    private Node<T> LLRotate(Node<T> avlNode)//传入要LL的node A
    Node<T> node=avlNode.leftChild //node B
    avlNode.leftChild=node.rightChild;//A的左节点->BR
	node.rightChild=avlNode //B的右节点->A
    return node;
}

private Node<T> rrRotate(Node<T> avlNode)//传入A
    Node<T> node=avlNode.rightChild;
	avlNode.rightChild=node.leftChild;
	node.leftChild=avlNode;
	return node;
}
